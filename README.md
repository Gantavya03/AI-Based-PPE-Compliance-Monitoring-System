# AI-Based PPE Compliance Monitoring System

An AI-powered industrial safety monitoring system developed using YOLOv8, ByteTrack, Flask, MySQL, and Raspberry Pi.

## Features

* Real-time PPE detection
* Worker tracking using ByteTrack
* Flask dashboard for monitoring
* MySQL database logging
* Screenshot evidence storage
* Raspberry Pi based alarm system
* Physical acknowledgement button
* LED and buzzer alerts

## Technologies Used

* Python
* OpenCV
* YOLOv8
* Flask
* MySQL
* Raspberry Pi
* RPi.GPIO

## Hardware Used

* Raspberry Pi 3B+
* LED
* Passive Buzzer
* Push Button
* Webcam/CCTV
